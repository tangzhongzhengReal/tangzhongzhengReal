首先介绍一个 github 仓库：v8-pref

与 V8 有关的注释和资源，以及与 Node.js 性能有关的资源。

主要介绍的几个专题有：数据类型，编译器，垃圾回收，内存分析等。每个专题下的文章都会有很多关联的资源。

第二个是 Google 工程师的演讲 Google I/O 2012 - Breaking the JavaScript Speed Limit with V8

这个演讲比较早期了，就所有关于 Hidden Class 的文章不已例外的也都引用了这次分享，虽然部分内容已经过时了，但是作为了解 V8 的设计思想还是很不错的。
