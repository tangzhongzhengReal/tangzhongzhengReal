Taro 引入全局变量

"@tarojs/plugin-sass": "^1.2.26"

在 config 中引入 path 组件

在 plugin 中添加 sass

resource 是指的 scss 文件的地址，绝对路径。
projectDirectory 是指项目的地址，绝对路径。

全面屏的适配

因为这次是自定义的 tabbar 和导航栏，所以上下都要考虑安全距离。
首先看导航条：
导航条由两部分组成：

第一部分是显示时间那里，叫做 statusBarHeight，这个高度每个手机都不同。可以使用 wx.getSystemInfoSync() 接口来获得。
第二部分是胶囊按钮的高度，在 ios 上是 40px，安卓上是 48px，而在模拟器上是 44px。

底部 tabbar 的安全距离应该是多高，没有官方的接口可以获取，为了兼容 iphoneX 系列和安卓的全面屏手势，应该给屏幕下面留出一定的高度来响应手势操作。这里我选择了屏幕高度超出 750px 就按照全面屏来处理，因为 iphone plus 系列的高是 738，x 的高度要高过 750。
