微信小程序请求封装和登录流程

官方流程图片

  1. 使用 wx.login()拿到 code
  2. 将 code 发送给后端，后端用 appid 等和微信服务器换取 openid
  3. 后端将 openid 和 session 或者 token 做关联，将 token 返回前端
  4. 前端再请求时携带 token 鉴权。
  5. 就是上面这个流程

怎么无缝做到这个过程

  1. 前端发送请求时后端判断，如果未登录则返回约定状态码
  2. 遇到约定状态码时，调登录接口。
  3. 登录成功之后，重新发起请求。
