Attr、Text 和 Comment, Element 上的一些方法

childNodes 能获取到什么

当我们拿到一个 div 元素节点的 dom 对象，一个 HTMLDElement 类的对象时，我们使用 Node 类原型上的 childNodes 属性，可以获取到他的子节点。在返回的集合里，有 Element，Comment 以及 Text 类对应的节点类型。这个属性名看着是获取所有的节点信息，但是只会返回这三种，Attr 属性节点都不会返回。

只有通过 Element. attributes 属性 能获取该元素节点的所有属性子节点。

我们知道 Document 对象也继承了 Node 类，所以 document 对象也可以使用 childNodes，这时候一般返回 doctype 那个文档节点和 html 元素节点。

element 的一些方法

取节点相关

  1. chlidNodes
  2. children 只返回元素节点
  3. firstChild
  4. firstElementChild
  5. lastChild
  6. lastElementChild
  7. nextSibling
  8. nextElementSibling
  9. previousSibling
  10. previousElementSibling
  11. parentNode. 返回一定是一个 Element
    取属性
  12. className 返回字符串
  13. classList 数组
  14. style 返回所有的 style 属性和值
    宽高度相关 height 和 width，offset 可理解为补偿？偏移？
  15. clientHeight 自己实际的高度，不含边框和滚动条
  16. offsetHeight 含类边框和滚动条的高度
  17. scrollHeight 内容的大小，即刚好不显示滚动条的大小
    Top Left 相关
  18. clientTop border-top 的厚度
  19. offsetParent 定位父级
    1）如果流式布局下来的话，那么一个元素的 offsetParent 是 body，body 的 offsetParent 是 null
    2）如果他是 fixed 的话，那么该元素 offsetParent 是 null
    3）总之，最外层 offsetParent 都是 null
    4）父级有 relative absolute 定位元素了，就找离他最近的。
  20. offsetTop 该元素 距离 offsetParent 元素顶部的高度，从自元素 border 外开始算到父元素 border 内结束。
  21. scrollTop 滚动条滚动的距离，a 元素内有滚动条，a.scrollTop=200 可以设置滚动条的滚动距离是向下滚了 200px
