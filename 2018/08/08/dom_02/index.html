    什么是 Element？

    Element 是什么

    上一节中我们知道了 document 是我们操作 DOM 的入口，一个 HTML 里只有一个 document 文档对象。

    我们知道 document 树中有元素节点，文本节点，属性节点，CSS 节点等。但是只有元素节点才能作为非叶节点。我们从 document 这个文档树中获取节点时，一般只获取到元素节点（也可以获取 doctype 以及 html 外的注释，没见过这么写的），我们会这么做：
    let adiv = document.getElementById('aid)

    对不对。当我们拿到 adiv 后，分析一下他的类型：
    adiv.__proto__

    会打印出 HTMLElement 或者是他的子类 xxxHTMLElement，之后对这个类进行查看，他的原型链是：
    HTMLElement->Element->Node->EventTarget

    Element 就是元素节点对应的类，HTMLElement 是对他之上做了一点对 HTML 的封装

    一点小差异

    Document 中有 getElementById，getElementsByTagName，getElementsByClassName 这三个获取元素节点的方法。

    Element 中有 getElementsByTagName，getElementsByClassName 这两个获取元素节点的方法。

    Document.getElementsByTagName !== Element.getElementsByTagName
    并且他们不相等，是各自的实现。
