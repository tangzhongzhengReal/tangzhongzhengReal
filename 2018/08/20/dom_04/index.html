Document 的一些方法，Node 的一些方法

Document 的作用主要是查找元素节点和创建元素节点

查找部分 api

getElementById getElementsByClassName getElementsByTagName

querySelector querySelectorAll 这两者按照 css 选择器来查找。

创建部分 api

  1. createElement()
  1. createDocumentFragment()
    当 js 频繁给 dom 做插入更新的时候，每一次都会重绘重排
    当创建一个文档碎片之后，将新的 Element 插入到文档碎片中，最后一次性插入到 dom 中，减少性能损耗。
  1. 文档碎片可以存储从 dom 中获取的节点，当存入文档碎片时，浏览器页面就不会再显示他了，通过之前的引用变量还是可以修改存入的节点。

Node 的主要作用就是插入移除节点。

appendChild. insertBefore. removeChild replaceChild

append prepend 可以传入多个 dom 或者 domString 可以同时插入最前或者最后。
