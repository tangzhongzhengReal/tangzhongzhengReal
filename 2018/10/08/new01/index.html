new 的作用

众所周知，我们新建一个类的实例的时候，会用 new Class()来实现，那么这么一句做了什么呢？
1）创建一个空的对象
2）把 Class 的实例对象取出来放到空对象上，并赋值。
3）把 Class 的 prototype 挂到之前创建的对象的__proto__上。
4）把 this 对象指到自身。

所以 new 主要做的就是创建了一个对象并且把各种属性绑到自己身上。
